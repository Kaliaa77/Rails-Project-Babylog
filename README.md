Babylog — Application d’accompagnement aux jeunes parents

  Babylog est une application pensée pour accompagner les jeunes parents d’enfants entre 0 et 4 ans.
  Elle combine un suivi personnalisé du profil de l’enfant avec un assistant IA capable de répondre aux inquiétudes du quotidien et de générer des plans d’action simples, en 3 à 4 étapes.

Ce projet a été réalisé en groupe dans le cadre du projet de fin de bootcamp.


🚀 Fonctionnalités principales :

👶 Gestion complète du profil bébé
  - Création d’un profil bébé avec informations détaillées
  - modification et suppression
  - Interface simple et intuitive

📖 Fonctionnalitées supplémentaires :
  - Suivi des vaccins de l'enfant sélectionné
  - Accès aux numéros d'urgences
  - Accèes aux gestes qui sauvent

💬 Discussion avec l’IA :
  - Page de chat personnalisé à l'enfant choisit pour poser des questions ou exprimer une inquiétude
  - L’IA analyse le message et génère : une réponse personnalisée, un plan de conseils structurés en 3-4 étapes
  - Objectif : rassurer, guider, accompagner les jeunes parents
  - Historique des conversations accessible


🔐 Authentification sécurisée
  - Gestion des utilisateurs via Devise
  - Sessions sécurisées

🛠️ Technologies utilisées
- Ruby on Rails 8	Framework principal
- PostgreSQL	Base de données
- Devise	Authentification
- Stimulus	Composants interactifs
- Bootstrap	UI / mise en forme
- ActionCable : Temps réel pour le chat
- API IA	Génération des réponses et conseils

📝 À propos
Babylog a été conçu pour offrir un soutien bienveillant aux jeunes parents, en combinant technologie moderne et conseils pratiques.
L’objectif : réduire le stress, répondre aux inquiétudes, et accompagner les familles dans les premières années de vie de leur enfant.


📦 Installation & Setup
1. Cloner le projet
Code
git clone https://github.com/Kaliaa77/Rails-Project-Babylog.git
cd Rails-Project-Babylog
2. Installer les dépendances
Code
bundle install
yarn install --check-files
3. Configurer la base de données
Code
rails db:create
rails db:migrate
4. Lancer le serveur
Code
rails s
L’application sera disponible sur :
👉 http://localhost:3000
